## 황인찬 / 고민하는 개발자

**"왜 이 기술을 선택해야 할까?"**

이 질문에서 모든 개발이 시작됩니다. 단순히 기능을 구현하는 것이 아니라, 
**사용자에게 좋은 서비스**를 만들고 싶어서 항상 고민합니다.

그 결과 검색 성능을 99.5% 개선하고, 서버 부하를 89.5% 줄이며, 
318명의 신규 사용자를 확보하는 성과를 거둘 수 있었습니다.

기술과 사용자 가치, 둘 다 놓치지 않는 개발자로 성장해나가고 있습니다.
